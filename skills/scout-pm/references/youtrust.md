# YouTrust 専門エージェント

YouTrust（ユートラスト）は「信頼でつながるキャリアSNS」を掲げるプラットフォーム。最大の特徴は**チャット形式**でのメッセージ送信と、**2次つながり（友達の友達）までしかスカウトを送れない**という制約。IT・エンジニア系の人材が多く、技術的な面白さや成長機会への訴求が効果的。

**重要な制約：**
- ブラウザ自動操作・クローリングは利用規約で明確に禁止されている
- 自社以外の紹介・あっせんを目的としたスカウト送信は禁止とされているが、ヘッドハンティングとしての利用は個別確認が必要
- **2次つながりまでしか送れない**ため、母集団は他媒体より小さくなる

## 検索条件の生成ルール

### YouTrust検索の特性
- **スキルタグ**とフリーワード検索が中心
- 2次つながりの範囲内での検索となるため、母集団に限りがある
- 転職意欲ステータス（「積極的に探している」「良い話があれば」等）でフィルタ可能

### 広め条件

出力フォーマット：
```
■ YouTrust 広め条件

スキルタグ: ◯◯, ◯◯（大きめのカテゴリ）
フリーワード: ◯◯
転職意欲: 指定なし（全ステータス）
つながり: 2次つながりまで

想定ヒット感: つながりの範囲に依存。まず母集団の規模を確認する。
```

### 絞り込み条件

出力フォーマット：
```
■ YouTrust 絞り込み条件

スキルタグ: ◯◯, ◯◯, ◯◯（具体的なスキル）
フリーワード: ◯◯（具体的な経験・職種キーワード）
転職意欲: 「積極的に探している」「良い話があれば」

想定ヒット感: かなり少なめ。つながりの範囲と転職意欲フィルタで絞られる。
```

### 検索条件のポイント
- スキルタグはYouTrustに登録されているタグに合わせる（正確なタグ名は媒体上で確認が必要）
- IT・エンジニア系のスキルタグが充実している（Ruby, Python, AWS, プロダクトマネジメント等）
- 2次つながりの範囲が狭い場合は、まずつながりを広げる活動が先に必要な場合もある

## スカウト文面の生成ルール

### YouTrust向けの特性

| 項目 | ルール |
|------|-------|
| 日程調整URL | **含める**（SpirInc URL） |
| トーン | チャット形式。カジュアル寄りだがビジネスの最低限の丁寧さは保つ |
| 長さ | **250〜400文字** |
| フォーマット | プレーンテキスト。チャットの自然なテンポを意識し、段落を短く |
| 言語 | 日本語 |

### 文面構成のポイント

YouTrustはチャット形式のプラットフォーム。メールのような長い文面は場違いに見える。IT・エンジニア系の人材が多いので、技術的な具体性を出すと響く。

効果的なアプローチ：
1. **「はじめまして」や「初めてご連絡いたします」で入る**（「お疲れ様です」は知り合い同士の表現なので初対面では使わない）
2. 簡潔な自己紹介（1行）
3. 候補者のスキル・経験に触れた着目ポイント（技術的な具体性を出す）
4. ポジションの魅力（技術的チャレンジや成長機会を中心に）
5. 日程調整URLで締める
6. 段落を短くし、チャットの読みやすさを意識

### テンプレート

```
{候補者名}さん

はじめまして！
合同会社Plus X代表の藤本です。
興味深いポジションの紹介に強みを持つヘッドハンティング会社です。

{候補者名}さんのプロフィールを拝見して、ぜひご紹介したいポジションがあります。

{着目ポイント_短縮版：技術的な具体性を出した1〜2行}

【ポジションの面白さ】
◎（技術的チャレンジ・成長機会の観点で）
◎（事業の面白さを企業名を伏せて）
◎（条件面を簡潔に：年収◯◯万〜 / リモート◯◯）

もしご興味あれば、30〜60分ほどお話しできると嬉しいです。
https://app.spirinc.com/patterns/availability-sharing/9782b79f-044a-44fc-8cb3-98cef3d07975/confirm

お気軽にご返信ください！
```

### 絶対に守ること
- チャット形式のトーンにする（メール文面のような堅さにしない）
- **「お疲れ様です」は使わない**（初対面では不自然）
- 段落を短く、テンポよく
- 技術的な専門用語を適切に使う（「データ基盤」「マイクロサービス」「グロース」等、職種に合ったワード）
- 企業名・プロダクト名は絶対に含めない
- 400文字を超えない

## ブラウザ操作ワークフロー（Claude in Chrome）

YouTrustでの候補者リストアップをブラウザ上で行う場合の手順。
Claude in Chrome のツールを使って以下の順序で操作する。

**URL：** https://admin.youtrust.jp/headhunter/search_users

### 手順

1. **検索画面を開く**
   - 上記URLにアクセスする

2. **検索条件を入力して絞り込む**
   - スキルタグ、フリーワード、転職意欲などのフィルタを、このリファレンスで生成した検索条件に基づいて設定する
   - まず広め条件で母集団を確認し、多すぎれば絞り込み条件に切り替える

3. **候補者ごとに「気になる」ボタンを押す**
   - 検索結果の候補者一覧から、マッチ度が高そうな候補者の「気になる」ボタンを押す
   - 「気になる」を押すと、リスト選択の画面が表示される

4. **「新規リストを作成」で求人用のリストを作成する**
   - 最初の候補者で「気になる」を押した際、「新規リストを作成」ボタンをクリック
   - リスト名は **求人のポジション名**（例：「PdM_テックフォワード案件」）で作成する
   - 以降の候補者は、作成済みのこのリストに追加していく

5. **候補者をリストに追加し続ける**
   - 検索結果を順に確認し、マッチ度が高い候補者ごとに「気になる」→ 作成済みリストに追加
   - 評価フレームワーク（マッチ度4以上）を参考に、スカウトを送る価値がある候補者を選定する
