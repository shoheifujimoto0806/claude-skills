# スカウトメール共通テンプレート

媒体ごとのトーン調整は scout-operator が行う。ここでは共通ベースの文面構造を定義する。

## 基本構造

```
{候補者名}様

はじめまして。
合同会社Plus X代表の藤本と申します。

弊社は興味深いポジション・ビジネスを有している企業の
紹介に強みを持つヘッドハンティング会社です。

ご経験を拝見し、
ぜひとも【ポジション名（企業名を含めない形で記載）】をご紹介させていただきたく、
ご連絡させていただきました。

【着目させていただいたポイント】
{着目ポイント}
▶︎上記のご経験から、本ポジションにおいて大きな成果を発揮いただけると考えております。

【求人の魅力】
◎（企業名を伏せた形で、ポジション・事業の魅力を3点程度）
◎
◎

もし少しでもご興味をお持ちいただけましたら、
求人のご案内のお時間を頂戴できないでしょうか。

{日程調整セクション}

是非とも少しでもお話をするお時間を頂戴できますと幸いです。
ご返信いただけることを心よりお待ちしております。
```

## 日程調整セクション

日程調整セクションの内容は各媒体の platform.md を参照すること。

## プレースホルダー一覧

| プレースホルダー | 説明 |
|----------------|------|
| `{候補者名}` | 候補者の氏名 |
| `{着目ポイント}` | マッチ度4以上の候補者に対して作成する着目ポイント（3点程度） |
| `{日程調整セクション}` | scout-operator が媒体の platform.md に基づいて埋める |
