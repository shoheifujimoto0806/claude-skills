# YouTrust 操作リファレンス

YouTrust（ユートラスト）は「信頼でつながるキャリアSNS」を掲げるプラットフォーム。IT・エンジニア系の人材が多く、技術的な面白さや成長機会への訴求が効果的。最大の特徴は**チャット形式**でのメッセージ送信。

## 重要な制約

- ブラウザ自動操作・クローリングは利用規約で明確に禁止されている
- 自社以外の紹介・あっせんを目的としたスカウト送信は禁止とされているが、ヘッドハンティングとしての利用は個別確認が必要
- **2次つながりまでしか送れない**ため、母集団は他媒体より小さくなる

## 検索

### 検索画面

- URL: `https://admin.youtrust.jp/headhunter/search_users`
- ログイン済みブラウザが前提

### フィルタ操作

検索画面の左側にフィルタパネルがある。以下の順序で設定する:

**転職意欲**（最重要フィルタ）
- ドロップダウンで選択。推奨: 「まったく考えていない」以外を選択
- 「積極的に検討している」「検討している」は返信率が高い

**職種**
- ドロップダウンで選択。prep.yaml の target_titles に近いカテゴリを選ぶ
- サブ職種でさらに絞り込み可能

**フリーワード**
- テキスト入力欄に検索ワードを入力
- AND/OR の切り替えが可能（デフォルト: AND）
- スペース区切りで複数ワード指定可能（例: 「フルスタック React TypeScript」）
- prep.yaml の broad_keywords → narrow_keywords の順で試す

**その他フィルタ**
- 副業意欲: 副業案件の場合のみ使用
- 希望の働き方: リモート/オフィスの指定がある場合
- 希望の働く場所: 勤務地の指定がある場合
- 最終ログイン: 「1週間以内」等でアクティブユーザーに絞る
- 経験社数・卒業年度: 通常は指定なし

**除外項目**
- 「公式リクルーターを除く」: チェック推奨
- 「スカウト済みのタレントを除く」: 重複防止にチェック推奨
- 除外ワード: 不要な職種キーワード等を入力

**スキルタグ**（フリーワードとは別）
- YouTrust に登録されているタグから選択（Ruby, Python, AWS, プロダクトマネジメント等）
- 正確なタグ名を使う必要がある

### ページネーション

- 検索結果が複数ページにわたる場合、ページ送りで遷移
- 最終ページの判定: 「次へ」ボタンが無効化または非表示になったら最終ページ

### 検索のコツ

- まず広めのスキルタグで母集団の規模を確認する
- つながりの範囲が狭い場合は、先につながりを広げる活動が必要な場合もある
- スキルタグは媒体上に登録されている正確なタグ名を使う

## リスト管理

### リスト作成

- 候補者の「気になる」ボタンを押すとリスト選択画面が表示される
- 「新規リストを作成」ボタンでリストを作成
- リスト名は求人のポジション名で作成する（例: 「PdM_テックフォワード案件」）

### 候補者追加

- 検索結果から候補者ごとに「気になる」ボタンを押す
- 作成済みリストを選択して追加
- マッチ度4以上の候補者を優先的に追加する

### 候補者削除

- リスト画面から対象候補者を選択して削除

## セッション管理

- セッション切れの検知: ログインページ（`/login` や `/sign_in`）へのリダイレクトで判断
- セッション切れ時: ユーザーに再ログインを依頼し、完了後に操作を再開

## スカウト文面の特性

| 項目 | ルール |
|------|-------|
| 日程調整URL | 含める（SpirInc URL） |
| トーン | チャット形式。カジュアル寄りだがビジネスの最低限の丁寧さは保つ |
| 長さ | 250〜400文字 |
| フォーマット | プレーンテキスト。チャットの自然なテンポを意識し段落を短く |
| 言語 | 日本語 |

### 文面のポイント

- 「はじめまして」や「初めてご連絡いたします」で入る（「お疲れ様です」は知り合い同士の表現なので初対面では使わない）
- 段落を短くし、チャットの読みやすさを意識
- 技術的な専門用語を適切に使う（「データ基盤」「マイクロサービス」「グロース」等）
- 企業名・プロダクト名は絶対に含めない
- 400文字を超えない

### 共通テンプレートからの変換ルール

共通テンプレート（scout-message.md）から YouTrust 向けに変換する際のルール:

| 共通テンプレートの表現 | YouTrust 向けの表現 |
|---------------------|-------------------|
| 「〜させていただきました」 | 「〜しました」 |
| 「ご連絡させていただきました」 | 「ご連絡しました」 |
| 「頂戴できないでしょうか」 | 「いただけると嬉しいです」 |
| 「心よりお待ちしております」 | 「お待ちしています！」 |
| 長い敬語の段落 | 短い文で区切る |

原則: ビジネスの最低限の丁寧さは保ちつつ、チャットの自然なテンポを優先する。

共通テンプレートを YouTrust のチャットトーンに調整した例。

```

{候補者名}さんのご経験を拝見して、

【着目ポイント】
{着目ポイント}

◎（企業名を伏せた形で3点程度）
◎
◎

```

## 既知の制約・注意事項

- ページネーション時に最終ページの判定を誤りやすい
- リスト名に日本語を使う場合の文字化けに注意
- 検索結果0件の場合は条件を緩めて再検索する
