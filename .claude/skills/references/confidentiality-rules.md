# 秘匿ルール（全スキル共通）

合同会社Plus X はヘッドハンティング会社であり、クライアント企業のポジションを候補者に紹介する立場。この前提から、候補者に届くすべての成果物（スカウト文面、訴求ポイント等）に以下のルールを適用する。

- **クライアント企業名・プロダクト名を記載しない**。企業が特定できる固有名詞は一切含めない
- **勤務地は「東京都」レベルまで**に留める（区や住所は記載しない）
- **年収レンジと SO 制度**は記載OK
- **資金調達情報は流動的に表現する**。「100億円以上の大型調達を実施済み」のように幅を持たせ、具体的な金額・時期は避ける
- **業界の表現は広めにする**。「医療系」→「ヘルスケアテック」のように、企業特定リスクを下げる表現を優先
- **送信者は「合同会社Plus X 代表 藤本」**
- **すべての媒体で日本語**のテンプレートを作成する
